PyCode Conference 2019

PyCode Conference 2019 is a conference devoted to the Python language. The organizers are people from the language community.

The aim of the conference is to raise the participants' qualifications, show the latest applications in Python. It is also important for us to spread good programming practices.

The conference is addressed to all Python programmers. The PyCode Conference opens up to the community at a fairly wide level, which means it is intended for specialists of the so-called Python pokers as well as for people who are just starting to program.
